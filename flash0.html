ESP32 Wi-Fi monitor

Monitor deauth activity with a sharper UI.

Deauth Sleuth is a touchscreen ESP32 Wi-Fi monitoring project for the ESP32-2432S028R. It watches nearby 802.11 traffic in promiscuous mode, highlights deauthentication and disassociation activity, and presents live status with touch controls, alert visuals, SD logging, and RGB LED feedback in a cleaner futuristic layout.

Deauth Sleuth firmware

Install directly from the real Deauth Sleuth github repo for CYD and CYD2USB builds.
USB setup

Use a data-capable USB cable and close Arduino IDE, serial monitors, or any app already holding the COM port before using the web flasher.

Browser support

For web flashing, use a current Chromium-based browser such as Chrome or Brave. Other browsers may not support the required Web Serial features.
